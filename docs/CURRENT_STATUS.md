## Known limitations

- Android and iOS platform folders are generated by the bootstrap scripts using the installed Flutter SDK and are not committed in this first foundation branch.
- No `pubspec.lock` exists until `flutter pub get` is run by CI or locally.
- No Supabase project credentials are configured.
- The application displays preview data only; it does not read or write visitor records.
- Disabled buttons are intentional milestone boundaries, not completed features.
